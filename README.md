# OpenTripod
An open source tripod for Astronomy

# Main Specs
  - As parametric as possible (as my cad skills can handle)
  - Easy Pier Extension Mounting
  - Rope-based leg system
  - Extremely Robust (Metal on all load-bearing parts)
